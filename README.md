# Tian Dao 天道

A cultivation-themed idle RPG. Mine, cultivate, fight, trade.

## Stack

- Next.js 15 (App Router)
- Supabase (Postgres + Auth)
- Tailwind CSS + shadcn/ui
- Vercel (hosting)

## Local development

```bash
npm install
cp .env.example .env.local  # fill in Supabase keys
npm run dev
```
